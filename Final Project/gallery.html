Cheech's Flexible Pizza Parlor

The Cheech's Flexible Pizza Parlor project, I had to demonstrate flexing with flex boxes and media queries for responsive design without using positioning and floating. This was one of my favorite projects because I learned you will never need to float an element again. I was also enthused by the project because this was my first time learning about a flex box! It was a treat! I had to create two pages, an order and menu page for a pizza parlor with an order form and social media links. The menu page had required 6 flexible images to display 6 pizzas and a break point at 578 pixels where both pages had to stack in a mobile viewport.

In these labs I had to demonstrate different styles of using media queries and flexboxes without floating or using positioning.

The first demonstration were to create a parent div and 20 children divs inside the parent; wrap and flex the display. assign growing property of 1 for divs 1-19 and a property of 2 for the last div then flex the basis, height, and margin.

The second lab was to demonstrate three divs setting the width and height then assigning a specific flex growth property for each box.

The third lab was to demonstrate a pattern of wrapping and flex direction by setting three rows, each box set to specific widths to complete the row for 100% coverage horizonally.

The last lab demonstrated how to flex with three different media queries. first with a mobile version stacking flex boxes. the second query flexing the growth of the second box then wrapping the third and forth box next to each other then expanding the first and the fifth box. the third query was to expand the main content container with the text then stack all the flex boxes in a specific pattern and order.
